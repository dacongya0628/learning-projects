# 大葱鸭随机抽奖机

基于 Vite + TypeScript 的单页随机抽奖：支持自定义号码区间、多个中奖等级及各级人数、**不放回抽取**（同一人不会中两次），中奖号码按等级顺序 **一个一个** 揭晓，并带有滚动闪动与落定动画。界面为响应式布局，适配手机与 PC。

### 发给他人本地使用（Windows / macOS）

若要把本程序给别人在自己电脑上打开，请让对方阅读：**[使用者说明.md](./使用者说明.md)**（含 Node、仅 `dist`、Python 等多种方式，Windows 与 Mac 均适用）。

### 开发与源码说明

维护、二次开发或理解实现细节请参阅：**[开发说明.md](./开发说明.md)**（架构、模块职责、抽奖算法、构建原理等）。

## 使用方式

### 环境要求

- [Node.js](https://nodejs.org/) 18+（建议 LTS）

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

在浏览器中打开终端里提示的本地地址（一般为 `http://localhost:5173`）。

### 生产构建

```bash
npm run build
```

构建结果在 **`dist/`** 目录：主程序已打成一个 **`index.html`**（脚本与样式内联），**可直接双击 `dist/index.html` 在本地打开**（Windows / macOS 均可），也可把整个 `dist` 文件夹部署到任意静态站点。发给他人时建议 **ZIP 打包整个 `dist` 文件夹**。

本地用 HTTP 预览构建结果：

```bash
npm run preview
```

## 功能说明

1. **号码范围**：设置「最小号码」「最大号码」（整数闭区间），所有候选号码构成奖池。
2. **等级与人数**：可添加多行等级，每行填写名称与中奖人数；用 **↑ / ↓** 调整抽取顺序（先抽上面的等级）。
3. **校验**：各级人数之和不能超过区间内号码总数，否则会提示错误且无法进入舞台。
4. **两步流程**：先在「抽奖设置」中配置，点击 **「进入抽奖舞台」** 后再看到舞台与中奖记录；抽奖结束或任意时刻可点 **「返回抽奖设置」** 回到第一步并清空本轮结果。
5. **布局**：宽屏（约 768px 及以上）下 **抽奖舞台** 与 **中奖记录** 左右并排；窄屏下自动改为上下排列（舞台上、记录下）。
6. **抽奖过程**：在舞台页点击「开始抽奖」后，按等级依次抽取；每一注先播放滚动动画再显示真实号码；中奖记录会同步更新。
7. **重置本轮**：在舞台页可「重置本轮」清空记录并中止进行中的抽奖；设置页的「恢复默认设置」只恢复表单默认值。

## 项目结构

- `index.html` — 页面入口
- `src/main.ts` — 流程控制与事件绑定
- `src/state.ts` — 类型与默认等级
- `src/draw.ts` — 校验、洗牌与不放回队列
- `src/ui.ts` — DOM 结构、表单读写、揭晓动画与结果渲染
- `src/styles.css` — 可爱风样式与动画
- `src/version.ts` — 页脚展示用版本号（发版时请与 `package.json` 的 `version` 同步修改）
- `开发说明.md` — 架构、源码与算法说明（开发者阅读）
- `使用者说明.md` — 最终用户本地运行说明
